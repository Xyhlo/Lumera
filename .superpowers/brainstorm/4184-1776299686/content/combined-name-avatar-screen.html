Design Notes

  • Avatar preview + name input live at the top — always visible as you browse
  • Selected avatar updates the large preview in real-time with focus glow
  • Categories scroll horizontally (TV D-pad friendly), vertical navigation between rows
  • Focused avatar gets the theme's primary color border + glow (shown in cyan above)
  • Name field auto-focuses when screen opens — type immediately, then D-pad down to avatars
  • "Upload Your Own" accessible via a button at the end of any row or via remote shortcut
  • Replaces the old separate Name step + Avatar step into one unified screen
